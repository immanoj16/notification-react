# Notification-React

### It is an application which is built using React, Redux, Firebase.

  * Clone this repo
  ```
  git clone https://github.com/immanoj16/notification-react.git
  ```
  * install all dependencies
  ```
  yarn
  ```
  or
  ```
  npm install
  ```
  * start the app
  ```
  yarn start
  ```
  or
  ```
  npm run start
  ```
